# DoctorCode SDK for PHP/Laravel

AI-powered error detection and fixing for PHP applications. Automatically captures errors and provides intelligent fixes using Claude Sonnet 4, GPT-4o, Grok, or Gemini Pro.

## Features

- **Automatic Error Detection** - Captures all PHP errors in real-time
- **AI-Powered Fixes** - Get intelligent fix suggestions from multiple AI providers
- **Zero Configuration** - Works out of the box with Laravel
- **Privacy First** - Bring your own API keys for full control
- **Multi-LLM Support** - Choose from Claude, OpenAI, Grok, or Gemini
- **Smart Analytics** - Track error trends and fix success rates

## Installation

Install the package via Composer:

```bash
composer require doctorcode/sdk
```

## Quick Start

### 1. Get your API key from [https://doctorcode.dev](https://doctorcode.dev)

### 2. Add to `.env`:

```env
DOCTORCODE_API_KEY=your-api-key-here
```

### 3. Done!

All errors are now automatically reported to DoctorCode and analyzed by AI.
